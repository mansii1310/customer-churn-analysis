# Customer Churn Prediction

## Objective
Predict whether a customer will leave the service based on usage and contract details.

## Tools Used
- Python
- Pandas
- Seaborn
- Matplotlib
- Scikit-learn

## Machine Learning Model
- Logistic Regression

## Steps Performed
1. Data preprocessing and encoding
2. Exploratory data analysis
3. Train-test split
4. Model training
5. Model evaluation

## Evaluation Metrics
- Accuracy
- Precision
- Recall
- Confusion Matrix

## Business Impact
Helps companies reduce customer loss and improve retention strategies.
